# Sales Dashboard 📊

A Power BI dashboard analyzing sales performance, profit trends, and product/customer insights.

![Sales Dashboard](assets/Sales_Dashboard.png)

---

## Overview

This dashboard provides a consolidated view of sales and profit performance across states, products, and time, helping identify top-performing regions, products, and customers.

## Key Visuals

- **Sum of Profit by State** — Bar chart comparing profit across Karnataka, Maharashtra, Tamil Nadu, Kerala, and Telangana
- **Average of Profit by Product Name** — Area chart showing profit trend across products (Laptop, Phone, Tablet, Printer, Monitor, Headset, Keyboard, Mouse)
- **Average of Sales Amount by Product Name** — Bar chart of average sales value per product
- **Sum of Sales Amount by Year and Category** — Donut chart split by category (2026)
- **Sum of Profit / Count of Sales** — KPI gauge cards
- **Top Customer** — Highlight card for the highest-value customer
- **Month Slicer** — Interactive filter (January–June)

## Insights Covered

- State-wise profit distribution
- Product-wise average profit and sales performance
- Year and category-wise sales amount breakdown
- Identification of top customer by sales

## Tools Used

- **Power BI Desktop** — Data modeling, DAX measures, dashboard design
- **Excel** — Source dataset

## How to Use

1. Clone or download this repository
2. Open the `.pbix` file in Power BI Desktop
3. If prompted, update the data source path to point to your local copy of the dataset
4. Explore the interactive visuals and month slicer
